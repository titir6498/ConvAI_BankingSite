### Initial Setup
Ensure the following are installed on the target machine where code is to be run:
1. NodeJS (v18.0.0 or above)
2. React (v18.2.0)
3. Node Package Manager (npm)

### Install Project Dependencies
1. Navigate to /frontend/ folder and run 'npm install' using CLI
2. Navigate to /backend/ folder and run 'npm install' using CLI

### Launch App
1. Inside the /backend/ folder, run 'npm start' - API will be hosted on port 5000
2. Inside the /frontend/ folder, run 'npm start' - website will launch in a browser on port 3000
